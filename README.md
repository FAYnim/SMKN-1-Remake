## Overview

SMKN 1 Surabaya is a vocational high school (Sekolah Menengah Kejuruan) in Surabaya, Indonesia, offering various technical and business programs. This website serves as the official digital presence, providing comprehensive information about the school's profile, programs, departments, news, and announcements for prospective students, parents, and the public.

## Tech Stack

- **HTML5** - Semantic markup for all pages
- **CSS3** - Custom styles with CSS variables for theming
- **JavaScript** - Vanilla JS for interactive components
- **External Dependencies**:
  - Google Fonts (Bricolage Grotesque, Instrument Sans)
  - FontAwesome 6.4.0 for icons

## Features

- **Responsive Navigation** - Dropdown menus with mobile-friendly hamburger toggle
- **Smooth Interactions** - Smooth scroll behavior and transitions
- **Modern Design** - Editorial tech aesthetic with clean typography
- **Accessible** - Proper ARIA labels and semantic markup
- **No Dependencies** - Pure HTML/CSS/JS, no build process required

## Development

> [!TIP]
